
## Browser Support

- **iOS Safari 13+** — Full support with permission API
- **Android Chrome 80+** — Full support
- **Desktop Chrome/Edge** — Works but compass may be unavailable (shows manual entry)
- **Firefox** — Limited DeviceOrientationEvent support; may show fallback
